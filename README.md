# CRM
CRM project (just for sport)


## How to start the server

 - You need docker installed on your machine
 - Get into the server folder and run the following command
 ```bash
    docker-compose up -d
    yarn dev 
    or 
    npm run dev
 ```
 - You'll need to install pm2 to run prod version
 ```bash
    npm i -G pm2
 ```
 - then, run 
 ```bash
    npm run start 
    or 
    run start
 ```
